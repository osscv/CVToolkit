## ✨ 主要功能

### 🌐 网络工具（15 个）

- **速度测试** - 全面的网速测试，包含下载/上传速度、Ping 延迟、抖动测量和历史记录。支持前台服务不间断测试
- **Ping 测试** - ICMP Ping 测试，可配置包数、超时设置和详细统计（最小/最大/平均/丢包率）
- **持续 Ping** - 多主机实时 Ping 监控，带有彩色折线图，用于持续连接性分析
- **路由追踪** - 可视化网络路径追踪，显示所有跳转的延迟和地理信息
- **端口扫描** - 扫描任意主机的开放端口，支持服务检测和并行扫描
- **网络扫描（设备发现）** - 发现网络中所有连接设备，支持 MAC 地址查询和厂商识别
- **IP 查询** - 获取任意 IP 地址的详细地理位置、ISP、安全标识（VPN/代理/TOR 检测）和运营商信息，支持本地历史记录
- **DNS 查询** - 全面的 DNS 记录查询（A、AAAA、CNAME、MX、TXT、NS、SOA、PTR），支持自定义 DNS 服务器和响应时间追踪
- **DNS 基准测试** - 比较主流 DNS 提供商（Cloudflare、Google、Quad9、OpenDNS 等）的响应速度，找到最快的 DNS
- **子网计算器** - 高级 IPv4/IPv6 子网计算，支持 CIDR 表示法、网络/广播地址、可用主机范围和子网掩码转换
- **SSL/TLS 检查器** - 详细的 SSL 证书检查，包含有效期、颁发者信息、证书链、加密算法和过期警告
- **Whois 查询** - 获取域名注册信息
- **HTTP 头信息查看** - 检查 HTTP 响应头
- **自定义请求构建器** - 构建并发送自定义 HTTP/CURL 请求
- **WiFi 分析器** - 分析 WiFi 信号强度、频道和网络详情

## 🚀 开始使用

### 前置要求

- **Android Studio：** Ladybug (2024.2) 或更高版本
- **Android SDK：** API 24 (Android 7.0) 或更高
- **Kotlin：** 2.3+
- **Gradle：** 8.0+
- **JDK：** Java 11 或更高

### 安装步骤

1. **克隆仓库：**

```bash
git clone https://github.com/osscv/CV-Toolkit.git
cd CVToolkit
```

2. **在 Android Studio 中打开项目**
   - 选择"打开现有项目"
   - 导航到克隆的目录
   - Android Studio 会自动检测项目

3. **同步 Gradle 依赖**
   - Android Studio 会提示同步
   - 或手动操作：文件 → 与 Gradle 文件同步项目

4. **配置 AdMob（可选）**
   - 应用包含 AdMob 集成
   - AdMob ID 在 `AndroidManifest.xml` 中
   - 如果发布自己的版本请替换为您自己的 ID

5. **构建并运行**
   - 连接 Android 设备或启动模拟器
   - 点击运行 (▶) 或按 Shift+F10

### 构建命令

**调试构建：**
```bash
./gradlew assembleDebug
```

**发布构建：**
```bash
./gradlew assembleRelease
```

**安装到设备：**
```bash
./gradlew installDebug
```

**运行测试：**
```bash
./gradlew test
```

**APK 位置：**
- 调试版：`app/build/outputs/apk/debug/app-debug.apk`
- 发布版：`app/build/outputs/apk/release/app-release.apk`
